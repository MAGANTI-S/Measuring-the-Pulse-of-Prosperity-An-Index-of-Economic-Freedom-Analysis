Project Overview

This project analyzes global economic freedom using indicators such as unemployment, GDP growth, inflation, and public debt. The analysis helps understand national prosperity levels and economic performance across countries.

Tools Used

  • Tableau Public
  • Data Visualization Techniques
  • Economic Freedom Dataset

Key Insights

  • Countries with low unemployment tend to have higher economic freedom scores.
  • High inflation and public debt negatively impact economic performance.
  • Regional differences highlight global economic inequality.
