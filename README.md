# Measuring-the-Pulse-of-Prosperity-An-Index-of-Economic-Freedom-Analysis

## Project Overview
This project analyzes global economic freedom using indicators such as unemployment, GDP growth, inflation, and public debt.

## Tools Used
- Tableau Public
- Data Visualization
- Economic Freedom Dataset

## Key Insights
- Countries with low unemployment have higher index scores.
- High inflation and debt reduce economic performance.
- Regional differences highlight economic inequality.
